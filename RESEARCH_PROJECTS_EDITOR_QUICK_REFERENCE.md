## 💡 Tips for Using the Editor

1. **Save Frequently** - Use the Save buttons to preserve changes
2. **Format Before Save** - All formatting is retained when saved
3. **Upload Images** - Click upload icon to add images directly
4. **Preview Content** - The editor shows exactly how it will appear
5. **HTML Aware** - For advanced users, raw HTML can be edited in source view

## 🔧 Technical Details

- **Editor Type:** CKEditor 5 Classic
